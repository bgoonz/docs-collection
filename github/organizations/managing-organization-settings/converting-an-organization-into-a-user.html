Converting an organization into a user

{% ifversion fpt %}

  1. Sign up for a new GitHub user account.
  2. Have the user’s role changed to an owner.
  3. {% data variables.product.signin_link %} to the new user account.
  4. Transfer each organization repository to the new user account.
  5. Delete the organization.
  6. Rename the user to the organization’s name.

{% else %}

  1. Sign up for a new GitHub Enterprise user account.
  2. Have the user’s role changed to an owner.
  3. {% data variables.product.signin_link %} to the new user account.
  4. Transfer each organization repository to the new user account.
  5. Delete the organization.
  6. Rename the user to the organization’s name.

{% endif %}
