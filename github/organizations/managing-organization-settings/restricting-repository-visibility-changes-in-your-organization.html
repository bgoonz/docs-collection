Restricting repository visibility changes in your organization

You can restrict the ability to change repository visibility to organization owners only, or allow members with admin privileges for a repository to also change visibility.

{% warning %}

Warning: If enabled, this setting allows people with admin permissions to change an existing repository to any visibility, even if you do not allow that type of repository to be created. For more information about restricting the visibility of repositories during creation, see “Restricting repository creation in your organization.”

{% endwarning %}
