Changing a remote’s URL from GitHub Desktop

{% mac %}

  1. In the Repository menu, click Repository Settings….
  2. In the Primary remote repository field, type the desired URL.
  3. Click Save.

{% endmac %}

{% windows %}

  1. In the Repository menu, click Repository settings….
  2. In the Primary remote repository field, type the desired URL.
  3. Click Save.

{% endwindows %}
