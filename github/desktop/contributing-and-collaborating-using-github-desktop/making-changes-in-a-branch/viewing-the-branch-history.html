Viewing the branch history

Each commit shows:

  • The commit message
  • The time the commit was created
  • The committer’s username and profile photo (if available)
  • The commit’s SHA-1 hash (the unique ID)

{% data reusables.desktop.history-tab %} 2. On the History tab, click the commit you’d like to review. 3. If there are multiple files in the commit, click on an individual file to see the changes made to that file in that commit.
