Activity dashboard

The Activity dashboard provides weekly, monthly, and yearly graphs of the number of: - New pull requests - Merged pull requests - New issues - Closed issues - New issue comments - New repositories - New user accounts - New organizations - New teams

Accessing the Activity dashboard

  1. At the top of any page, click Explore.
  2. In the upper-right corner, click Activity.
