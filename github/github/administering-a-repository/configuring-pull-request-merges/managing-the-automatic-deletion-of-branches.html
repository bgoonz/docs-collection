Managing the automatic deletion of branches

Anyone with admin permissions to a repository can enable or disable the automatic deletion of branches.

{% data reusables.repositories.navigate-to-repo %} {% data reusables.repositories.sidebar-settings %} 3. Under “Merge button”, select or unselect Automatically delete head branches.
