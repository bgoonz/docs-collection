Creating new files

When creating a file on {% data variables.product.product_name %}, consider the following:

  • If you try to create a new file in a repository that you don’t have access to, we will fork the project to your user account and help you send a pull request to the original repository after you commit your change.
  • File names created via the web interface can only contain alphanumeric characters and hyphens (-). To use other characters, create and commit the files locally, then push them to the repository on {% data variables.product.product_name %}.
