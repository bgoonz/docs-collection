Finding files on GitHub

{% tip %}

Tips:

  • File finder results exclude some directories like build, log, tmp, and vendor. To search for files within these directories, use the filename code search qualifier.
  • You can also open the file finder by pressing t on your keyboard. For more information, see “Keyboard shortcuts.”

{% endtip %}

{% data reusables.repositories.navigate-to-repo %} {% ifversion ghes < 2.22 %} 2. Under your repository name, click Find file. {% else %} 2. Above the list of files, click Go to file. {% endif %} 3. In the search field, type the name of the file you’d like to find. 4. In the list of results, click the file you wanted to find.
