About data use for your private repository

When you enable data use for your private repository, you’ll be able to access the dependency graph, where you can track your repository’s dependencies and receive {% data variables.product.prodname_dependabot_alerts %} when {% data variables.product.product_name %} detects vulnerable dependencies. For more information, see “About alerts for vulnerable dependencies.”
