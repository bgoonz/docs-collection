Configuring webhooks for organization events in your enterprise account

You can configure webhooks to receive events from organizations owned by your enterprise account. For more information about webhooks, see “Webhooks.”

Next to “Webhooks”, click Add webhook. 5. Type a payload URL, then optionally customize the configuration. For more information, see “Creating webhooks.” 6. Under “Which events would you like to trigger this webhook?”, select Let me select individual events. 7. Select one or more enterprise account events for your webhook to receive. For more information, see “Event types and payloads.” 8. To receive the selected events for triggered webhooks, select Active. 9. Click Add webhook.
