Tip: When you make your project board {% ifversion ghae %}internal{% else %}public{% endif %}, organization members are given read access by default. You can give specific organization members write or admin permissions by giving project board access to teams they’re on or by adding them to the project board as a collaborator. For more information, see “Project board permissions for an organization.”

{% endtip %}
