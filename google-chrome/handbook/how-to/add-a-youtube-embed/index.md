## Add a YouTube Embed

To add an YouTube embed to a post you can use a custom shortcode.

{% raw %}

    {% YouTube
      id='whnms4CLJys'
    %}

{% endraw %}

{% YouTube id=‘whnms4CLJys’ %}

You can also specify the startTime in the second parameter.

{% raw %}

    {% YouTube
      id='whnms4CLJys',
      startTime='123'
    %}

{% endraw %}
