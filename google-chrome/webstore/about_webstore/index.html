What is the Chrome Web Store?

2015-03-21

The Chrome Web Store lets you publish extensions—either free or paid—where Google Chrome users can easily find them.

Your published item can be free, or you can charge for it using any payment system you like.

To upload and publish items, use the Chrome Developer Dashboard.

Why should I use the Chrome Web Store?

Publishing your extension in the Chrome Web Store lets you reach more people. You can also earn revenue or build Chrome-specific features into it.

How do I start?

  1. Learn more about Chrome Extensions
  2. Publish your first extension by following the steps in Publish in the Chrome Web Store.
  3. For addition help, check the Chrome Web Store FAQs
