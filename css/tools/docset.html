Docset

We use and ❤️ Kapeli’s Dash app for browsing API documentation. Dash comes with 150+ offline documentation sets, but doesn’t have our style guide.

So we’ve generated a GitHub Style Guide docset that gives you access to this site offline.

Installation

  1. First you’ll need to make sure you have Dash app installed.
  2. Once installed, open the app, and click this install docset link.
  3. After the docset has been added to your list, click “download”.
