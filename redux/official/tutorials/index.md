## <span id="redux-official-tutorials" class="anchor enhancedAnchor_2LWZ"></span>Redux Official Tutorials<a href="#redux-official-tutorials" class="hash-link" title="Direct link to heading">#</a>

The [**Quick Start** page](quick-start.html) briefly shows the basics of setting up a Redux Toolkit + React application, and the [**TypeScript Quick Start** page](typescript-quick-start.html) shows how to set up Redux Toolkit and React for use with TypeScript.

We have two different full-size tutorials:

- The [**Redux Essentials tutorial**](essentials/part-1-overview-concepts.html) is a "top-down" tutorial that teaches "how to use Redux the right way", using our latest recommended APIs and best practices.
- The [**Redux Fundamentals tutorial**](fundamentals/part-1-overview.html) is a "bottom-up" tutorial that teaches "how Redux works" from first principles and without any abstractions, and why standard Redux usage patterns exist.

##### <span class="admonition-icon"> </span>tip

**We recommend starting with the [Redux Essentials tutorial](essentials/part-1-overview-concepts.html)**, since it covers the key points you need to know about how to get started using Redux to write actual applications.
